📋 How to build this in Notion

Database Structure
  • Main DB: Engineering Log — Date, Type, Subsystem, Author, Status
  • Linked DB: Match Log — Match #, Event, Result, Score, Notes
  • Linked DB: Subsystems — Name, Owner, Progress, Status
  • Use Relations to connect entries → subsystems
Recommended Views
  • Dashboard: Gallery view of entries, sorted by date
  • Per-subsystem: Filtered board by Subsystem property
  • Timeline: Timeline view for build season calendar
  • My work: Filter by Author = your name
Each Entry Should Include
  • Goal: What were you trying to solve?
  • Process: What did you actually do?
  • Result: Did it work? Data/measurements?
  • Next Steps: What comes next?
  • Images: Photos, CAD screenshots, graphs
